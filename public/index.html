萌计数器！

使用说明

为您的计数器设置唯一ID，替换URL中的 :name 即可！

默认为随机主题，增加&theme=moebooru("moebooru"为指定主题)

列如：https://127.0.0.1/api/counter?name=1234&theme=moebooru

SVG地址
/api/counter?name=:name
图片标签
<img src="/api/counter?name=:name" alt=":name" />
Markdown
![\:name](/api/counter?name=:name)
BBCode
[img]/api/counter?name=:name[/img]

更多主题 ✨

使用查询参数 theme，例如：/api/counter?name=:name&theme=3d-num

工具

参数 说明 值
name 唯一计数器名称
theme 选择计数器主题
length 设置最小长度（1-16）
scale 设置图片缩放（0.1-2）
offset 设置像素偏移值（-500至500）
align 图片对齐方式
pixelate 像素化模式
darkmode 黑暗模式

特殊选项
base 基础数量
num 固定显示
